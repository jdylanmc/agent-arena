## Summary

Build the **minimum viable foundation** for Agent Arena: a VS Code extension at `extension/` that integrates `@github/copilot-sdk@^0.1.8`, registers an Activity Bar view containing a webview "primary agent terminal", and proves the end-to-end round-trip from user prompt в†’ SDK в†’ model в†’ streamed response. Wire EI-1 telemetry through the SDK's built-in OpenTelemetry pathway with a normalization layer feeding the canonical Agent Arena event log. Wire EI-2 by capturing per-session manifests + content hashes that reference the SDK's session-state directory. Ship a binary yolo toggle (default OFF, per-workspace) that gates the SDK's mandatory `onPermissionRequest` handler through a typed `PermissionPolicy` abstraction designed for future per-tool policies. Provide a CI matrix on `ubuntu-latest` + `windows-latest` that uses an `SdkAdapter` test seam with a `FakeSdkAdapter` (the SDK ships no consumer-facing test harness вЂ” research R-02). Land the wiki tree, two source-ingestion synthesis pages, and the log + harness schema docs as part of the same PR.

**Scale/scope**: one agent (the primary), one webview view, one round-trip path. The Swarm UI (multiple agents, workflow panel, background agents grid) is **out of scope** for this spec.
