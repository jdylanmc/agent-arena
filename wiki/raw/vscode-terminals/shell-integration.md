# VS Code Docs — Terminal Shell Integration

User-facing documentation for VS Code's shell-integration mechanism —
the substrate that powers command decorations, command navigation,
sticky scroll, quick fixes, run-recent-command, go-to-recent-directory,
and IntelliSense in the integrated terminal. Includes the `OSC 633`
custom escape-sequence catalog plus `OSC 133` (Final Term) and
`OSC 1337` (iTerm2) compatibility.
